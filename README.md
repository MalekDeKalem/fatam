fatam ist ein Tool für die Visulasierung von 3D oder 4D NIFTI Dateien
und kann auch DICOM-Series Daten visualisieren.


# How to run 
Holen Sie sich das Repository durch 
`git clone https://github.com/MalekDeKalem/fatam.git`

Gehen Sie in das Repository und erstellen eine virtuelle Umgebung.
Sie sollten dann die Umgebung aktivieren 
und die dependecies in requirements.txt mit pip installieren.

`cd fatam 
python -m venv env
source env/bin/activate
pip install requirements.txt
`

Welchen activate script man benutzen soll hängt vom Terminal ab 



# Daten 
In dem test Ordner gibt es eine NIFTI Datei, die man benutzen kann.
In dem dicomseries Ordner gibt es mehrere DICOM Dateien, die alle zu einer 
DICOM-Series gehören. Sie können auch importiert werden
